Private Sphere is a social blog theme for GatsbyJS created by [Chris Vogt](https://www.chrisvogt.me) for his personal website and blog.

Private Sphere comes with a social overview page with built-in GitHub, Goodreads, Instagram and Spotify components. It also comes with a blog index page and a default blog post template with built-in shortcodes.

### JSX in Markdown

This theme uses MDX, a Markdown format that supports JSX inside of documents. Use this to create and embed components with graphs and visualizations pertaining to the data in your article.

<HelloWorld />

### Embedded Content

Embed YouTube videos into posts using the `<YouTube />` shortcode.

```jsx
<YouTube url='https://www.youtube-nocookie.com/embed/aqz-KE-bpKQ' />
```

Which results in...

<YouTube url='https://www.youtube-nocookie.com/embed/aqz-KE-bpKQ' />
