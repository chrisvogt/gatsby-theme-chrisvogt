This document showcases features supported by the Private Sphere Gatsby theme. Private Spere is the Gatsby theme made for [my personal website](https://www.chrisvogt.me). The theme is built using:

- [Gatsby](https://www.gatsbyjs.org/) – open source blog framework based on React
- [React](https://reactjs.org/) – library for building UIs
- [MDX](https://mdxjs.com/) – Markdown format that supports JSX in documents
- [Theme UI](https://theme-ui.com/) – themable styles for React components
- [FontAwesome](https://fontawesome.com/) – icons for the web
